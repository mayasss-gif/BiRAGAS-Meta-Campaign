## Exporting Ad Images

1. Open any carousel/ad HTML file in Chrome
2. Right-click → Inspect → Toggle device toolbar
3. Set dimensions to 1080x1080 (or 1200x628 for link ads)
4. `Cmd+Shift+P` → "Capture full size screenshot"
5. Upload PNG to Meta Ads Manager
